Introduction to NER Disambiguation¶

Named Entity Disambiguation is the task of mapping words of interest, such as names of persons, locations and companies, from an input text document to corresponding unique entities in a target Knowledge Base (KB). Words of interest are called Named Entities (NEs), mentions, or surface forms.

NER Disambiguation model is built on top of NER model, which resolves the ambiguation based on the context.

In [6]:
#Initializing document assembler
document_assembler = DocumentAssembler()\
  .setInputCol("text")\
  .setOutputCol("document")

# Sentence Detector annotator, processes various sentences per line
sentence_detector = SentenceDetector()\
  .setInputCols(["document"])\
  .setOutputCol("sentence")

# Tokenizer splits words in a relevant format for NLP
tokenizer = Tokenizer()\
  .setInputCols(["sentence"])\
  .setOutputCol("token")

#Loading embeddings models from local disk
word_embeddings = WordEmbeddingsModel.load(pathbuilder.load_pretrained_model_path('glove_100d_en_2.4.0_2.4'))\
     .setInputCols("document",'token')\
     .setOutputCol("embeddings")

#Generation sentence level embeddings
sentence_embeddings = SentenceEmbeddings()\
  .setInputCols("sentence","embeddings")\
  .setOutputCol("sentence_embeddings")

#loading NER DL model from local Disk
ner_model=NerDLModel.load(pathbuilder.load_pretrained_model_path('ner_dl_en_2.4.3_2.4'))\
    .setInputCols(["sentence", "token", "embeddings"])\
    .setOutputCol("ner")

#Initializing NER converter
ner_converter = NerConverter() \
  .setInputCols(["sentence", "token", "ner"]) \
  .setOutputCol("ner_chunk")\
  .setWhiteList(["PER"])

#Initializing NER Disambiguator
disambiguator = NerDisambiguator() \
    .setS3KnowledgeBaseName("i-per") \
    .setInputCols("ner_chunk", "sentence_embeddings") \
    .setOutputCol("disambiguation") \
    .setNumFirstChars(5)

#nlppipeline for disambiguation
nlpPipeline = Pipeline(stages=[
    document_assembler, 
    sentence_detector,
    tokenizer,
    word_embeddings,
    sentence_embeddings,
    ner_model,
    ner_converter,
    disambiguator])

empty_data = spark.createDataFrame([[""]]).toDF("text")
model = nlpPipeline.fit(empty_data)
